# MTG Commander Banlists

Last updated: February 2026

## Commander/EDH Banlist

Source: https://mtgcommander.net/index.php/banned-list/

### Structurally Banned (All Formats)
- All ante cards (Contract from Below, etc.)
- Chaos Orb, Falling Star (dexterity)
- Shahrazad (subgame)
- All Conspiracy-type cards
- Cards with acorn stamp
- Offensive cards per WotC list

## Quick Check Function

To verify if a card is banned:
1. Check structurally banned first (card types, mechanics)
2. Check format-specific banlist
3. For Duel Commander, also check "banned as commander only" list
