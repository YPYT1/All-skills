# Commander Deck Templates

## Standard 100-Card Structure

### Balanced Template (Multiplayer)
```
Commander: 1
Lands: 36-38
  - Basic lands: 10-15
  - Dual/Fetch/Shock: 5-10
  - Utility lands: 5-10
  - Color fixing: 5-8

Ramp: 10-12
  - Mana rocks: 5-7 (Sol Ring banned in Duel)
  - Land ramp: 3-5
  - Mana dorks: 2-3

Card Draw: 10
  - Repeatable: 4-5
  - One-shot: 3-4
  - Cantrips: 2-3

Removal: 8-10
  - Single target: 5-6
  - Board wipes: 3-4

Protection: 3-5
  - Counterspells: 2-3
  - Hexproof/Indestructible: 1-2

Win Conditions: 3-5

Theme Cards: 25-35
```

### Duel Commander Template (1v1, 20 life)
```
Commander: 1
Lands: 33-36 (lower curve = fewer lands)

Ramp: 6-8 (Sol Ring banned!)
  - Mana dorks: 3-4
  - 2-mana rocks: 2-3
  - Land ramp: 1-2

Card Advantage: 8-10
  - Cantrips important: 4-5
  - Card draw: 4-5

Removal: 10-12 (1v1 needs more interaction)
  - Cheap removal: 6-8
  - Flexible answers: 4

Protection/Interaction: 5-7

Win Conditions: 2-3

Efficient threats: 30-35
```

## Archetype Templates

### Aggro (Duel Commander)
Focus: Fast damage, low curve
```
Lands: 32-34
1-drops: 10-12
2-drops: 12-15
3-drops: 8-10
4+ drops: 5-8
Removal: 8-10
Card draw: 5-7
```

### Control (Both formats)
Focus: Answers, card advantage, late game
```
Lands: 37-40
Counterspells: 8-12
Removal: 10-15
Card draw: 10-12
Win conditions: 3-5
Ramp: 8-10
```

### Combo (Commander)
Focus: Assemble combo, tutors, protection
```
Lands: 34-36
Combo pieces: 6-10
Tutors: 6-8
Protection: 6-8
Card draw: 10-12
Ramp: 10-12
Interaction: 8-10
```

### Midrange (Both formats)
Focus: Value, flexibility, threats
```
Lands: 36-38
Ramp: 10
Value creatures: 15-20
Removal: 8-10
Card draw: 8-10
Threats: 10-15
```

### Tribal (Commander)
Focus: Synergy, lords, tribal payoffs
```
Lands: 35-37
Tribal creatures: 30-35
Lords/Anthems: 5-8
Tribal support: 5-8
Card draw: 8
Removal: 6-8
Ramp: 8-10
```

### Voltron (Both formats)
Focus: Commander damage, equipment/auras
```
Lands: 35-36

Auras: 5-8
Protection: 8-10
Ramp: 10

```

## Mana Base Guidelines

### Color Distribution
- Mono: 25+ basics, utility lands
- 2-color: 15 each color sources, 8-10 duals
- 3-color: 12 each, heavy on duals/fetches
- 4-color: 10 each, rainbow lands crucial
- 5-color: Command Tower, City of Brass, Mana Confluence essential

### Staple Lands by Color

**Colorless Utility:**
- Command Tower
- Reliquary Tower
- Rogue's Passage
- Arcane Signet (rock)

**White:**
- Eiganjo, Seat of the Empire
- Emeria, the Sky Ruin

**Blue:**
- Otawara, Soaring City
- Mystic Sanctuary

**Black:**
- Takenuma, Abandoned Mire
- Cabal Coffers (with Urborg)

**Red:**
- Sokenzan, Crucible of Defiance
- Valakut, the Molten Pinnacle

**Green:**
- Boseiju, Who Endures
- Gaea's Cradle (expensive)

## Budget Considerations

### Budget Mana Base (~R$50-100)
- Focus on basics
- Tap lands (gain life duals)
- Pain lands
- Check lands
- Battle lands

### Mid Budget (~R$200-400)
- Add shock lands
- Some fetch lands
- Filter lands
- Fast lands

### No Budget
- All fetches
- Original duals
- Mana Crypt (if legal)
- Cradle/Coffers
